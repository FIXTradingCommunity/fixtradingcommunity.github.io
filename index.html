Welcome to the FIX Trading Community on GitHub

Overview

This site is supported by the FIX Trading Community. The purpose for this site is the development of open technical standards and supporting open source reference implementations of these technical standards. Our belief is that quality technical standards are best developed in an environment where competent people can contribute if they are so motivated in as unimpeded way as possible, while protecting an orderly standards development process.

Projects

There are several projects on GitHub ongoing at this time. Such as the Simple Binary Encoding standard and FIX Orchestra.
